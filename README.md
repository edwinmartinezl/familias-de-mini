# Familias de Mini

Catalogo publico de familias de minis organizadas por dominio.

Este repositorio reune familias que ya pasaron una compuerta de certificacion
y que pueden mostrarse como base confiable para la comunidad.

## Que es una familia

Una familia es un conjunto organizado de minis relacionadas por una misma
tecnologia o capacidad.

Cada familia puede incluir:

- minis
- flows base
- esquemas de entrada y salida
- metadatos tecnicos
- evidencia de certificacion

## Regla de publicacion

Solo entran a este repositorio familias certificadas.

Eso significa que la familia publica ya posee un inventario activo validado
y separado de cualquier inventario exiliado o experimental.

## Familias publicadas

- `ffmpeg`: procesamiento multimedia, conversion, extraccion, empaquetado y flujos de streaming.
