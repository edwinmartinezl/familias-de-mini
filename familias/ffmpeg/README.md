# FFmpeg

Familia de minis enfocada en procesamiento multimedia.

Esta publicacion expone la vista certificada activa de `ffmpeg`: conversion,
codificacion, decodificacion, remux, extraccion, filtros, subtitulos,
empaquetado y flujos de streaming.

## Resumen

- base tecnica: `FFmpeg`
- estado publico: `certified`
- minis certificadas activas: `48`
- flows base certificados activos: `48`
- minis exiliadas: `4`

## Que incluye esta carpeta

- `family.yaml`: metadata publica de la familia
- `family_blueprint.yaml`: blueprint tecnico base
- `skills_catalog.json`: catalogo certificado activo
- `base_flows.json`: flows base certificados activos
- `compatibility_graph.json`: conexiones validas dentro de la vista activa
- `skills/`: definiciones YAML de las minis publicadas
- `schemas/`: contratos JSON de entrada y salida
- `certificacion/resumen_certificacion.json`: resumen publico de certificacion

## Capacidades visibles

- conversion de audio, video e imagen
- codificacion y decodificacion
- remux de medios
- extraccion de audio e imagen
- mezcla, merge y filtros
- empaquetado HLS y DASH
- subtitulos quemados
- thumbnails y transformaciones de frame
- flujos de streaming

## Nota

La carpeta publica muestra solo el inventario activo aprobado para comunidad.
Las minis exiliadas no forman parte de esta vista.
